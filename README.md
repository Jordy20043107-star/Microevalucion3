## Unidad 2: La Representación del Conocimiento
```mermaid
mindmap
  root((Unidad 2: La Representación del Conocimiento))
    Bases formales
      Lógica proposicional
        Usa conectores (Y, O, NO)
        Limitada: sin relaciones entre objetos
      Lógica de predicados
        Usa variables y cuantificadores
        Representa objetos y relaciones
    Ingeniería del conocimiento
      Adquisición de conocimiento
      Representación formal
      Validación e inferencia
      Mantenimiento del conocimiento
    Trabajo del Ingeniero del Conocimiento
      Intermediario entre experto y sistema
      Responsabilidades
        Selección del problema
        Interacción con el experto
        Diseño de la base de conocimiento
        Validación y refinamiento
    Estrategias útiles
      Entrevistas
      Observación de expertos
      Cuestionarios y análisis de casos
    Cuello de botella
      Dificultad para extraer conocimiento humano
      Problema principal en IA simbólica
```

## Unidad 3: Los Sistemas Expertos, Definición y Conceptos
```mermaid
mindmap
  root((Unidad 3: Los Sistemas Expertos))
    Concepto de Sistema Experto
      Programa que razona con conocimiento experto
      Resuelve problemas como un humano experto
    Arquitectura básica
      Base de conocimiento
        Reglas, hechos y heurísticas
      Motor de inferencia
        Aplica reglas y deduce conclusiones
      Interfaz de usuario
        Comunicación con el sistema
      Módulo de explicación
        Justifica las conclusiones
    Tipos de conocimiento
      Superficial
        Basado en experiencia y heurísticas
      Profundo
        Basado en principios y causas
    Características distintivas
      Capacidad de explicar decisiones
      Razonamiento simbólico
      Independencia entre conocimiento y motor
    Tipos de sistemas expertos
      Diagnóstico
      Interpretación
      Predicción
      Planificación
      Monitoreo
```
